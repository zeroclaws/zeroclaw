# zeroclaw

Zeroclaw is a planned lightweight, open-source AI agent runtime for Linux/VPS machines.

Target: beginner-friendly setup, low-resource operation, CLI + temporary web dashboard + Telegram private chat.

Current status: design/spec phase.
